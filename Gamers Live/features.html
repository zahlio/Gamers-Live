Pages

Below you can see a image and a little description about some of the features that are included in Gamers Live, would you like a live preview then you can click here for a live demo.

  • Frontpage

    The front page shows all the online streaming featured streamers, and list the 5 highest ranked ones (as an admin you will give each featured streamer an level, and this level determines if they are showed first or last.)

  • Search

    With the integrated search bar at the top right corner of every page users will have no problem by finding exactly what they want. Whether its a specific game they wish to view or a specific streamer, the smart search helps them towards finding what they need.

  • User CP

    The user CP allows users to change their original stats and to view their subscribed channels and events. It is also here users can get in touch with the staff of you site by submitting a support ticket.


  • Chat Module

    We have built a custom made chat system that automatically enables chat on every channel! Both side my side with the live steam but also windowed mode are supported.

  • Chat Moderation

    We have also build a tool to manage the chat, here clients will be able to ban or add moderators to the chat of their channel.

  • Admin CP

    As an admin you can easily manage your website and see live statistics of what is going on. Banning and changing settings for each client is as simple as clicking a link on their channel page. Additional features are also available, like adding other admins, answering support tickets, adding games and much more.


  • Events

    All users can create events, and it is simple too. Users simpy has to select their title, message, image, game and event time and the event is ready to be shown. Admins can feature events so they are displayed on the front page.

  • Event Pages

    The event pages is where the magic happens, first of all user can subscribe to events so they will be notified when the event goes live. But admins can also feature the event to be displayed on the frontpage.

  • Ticket Support

    Client can create tickets that you and your support team can answer. It is easy to access and easy for both clients and staff to quick and fast response to the tickets. A knowledgebase system is also build in that allows you to create treads and guides for your users.

  • Channel Pages

    The channel is designed to be easy and clean. But still have all the features as a dedicated chat module, statistics, streamer information, ADS, tipping feature and costumizability. User can at any time change title, game details etc. Also custom images for each user is available, and users can upload offline img, avatar etc.

  • Partners

    You can set your users to partner, and make special deals with them. Gamers Live will automatically set payments that are due to partners. Partners could be famous streamers that use your service and get a cut of the money made by advertisements and tipping. Partners have access to a special partner CP where they can see and monitor all the information relevant as a partner.
